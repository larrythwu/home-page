Verilog Traffic Light Simulator

Project Overview

This project is a physical model that simulates traffic lights at an intersection using DE1-SoC FPGA and a PS/2 keyboard. The logic is implemented in Verilog, and the LEDs on the FPGA are used to represent the traffic lights. The keyboard acts as the pedestrian call button.

Features

  • Simulation of a 4-way intersection with traffic lights
  • Pedestrian crossing button functionality
  • Realistic timing sequences for traffic flow
  • Emergency vehicle override mode

Technologies Used

Verilog, DE1-SoC FPGA, PS/2 Keyboard Interface
