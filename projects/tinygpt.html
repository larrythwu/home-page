TinyGPT

This project is a minimal GPT2 replica based on nanoGPT, trained and fine-tuned for Chinese Q&A. For more details and code, check out the repo below.

TinyGPT
Model & Dataset
  • Model: GPT2 replica using PyTorch, 124M parameters
  • Dataset: Huggingface wiki-zh (all Chinese Wikipedia, 1.9B tokens), fine-tuned with silk-road/Luotuo-QA-A-CoQA-Chinese (4M tokens)
  • Q&A Format: Formatted as <|prompt|>...<|response|>...<|end|>
Training & Hardware
  • Hardware: Lambda Cloud 1xA1000 server
  • Training: 12 hours (~$20)
  • Reporting: WanDB

Analysis

The Q&A data (silk-road/Luotuo-QA-A-CoQA-Chinese) we used is mostly from GRE-style reading comprehension questions, many based on literature snippets rather than real-life articles. As a result, many responses are fictitious or not factually accurate.
